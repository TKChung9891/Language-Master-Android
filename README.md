# Language-Master-Android
 
Language Master is a Android App for learning Japanese and Korean. This Language Master App mimics how children learn native languages by listening and speaking.  It helps users learn Japanese and Korean from beginner to expert levels.  This App uses Google Translate, Google Text-to-Speech (TTS) and Google Speech Recognizer in Java done in Android Studio.  

You can watch the demo of this App at https://youtu.be/pdALxU6beo8
